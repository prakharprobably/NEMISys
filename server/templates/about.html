about://NEMISys

NEMISys stands for Networked Event Management and Integration System
It is an event management solution developed in house by the InfoTech department @ Cambridge School, Noida, with a host of automation features that made organizing TechnOlympics 2.0.25 a breeze
Some of these are-

  1. Automatically sorting the Google Forms response spreadsheet into participant-wise tables and adding them to a centralized database
  2. Providing an API endpoint for integration with the TechnoBot discord bot for fetching userdata, and the website for publishing results automatically
  3. Making a single centralized database to reduce manual labor required when managing with pen and paper
  4. Providing a simple and intuitive UI to interact with said centralized database
  5. An account system based on the PoLP, with token based authentication, and self hosted role protected resources to prevent internal sabotages
  6. Providing a simple interface for updating registration attendance, and generating event-wise attendance tables from it
  7. Providing Event Incharges with a single interface to mark attendance and results of their respective events
  8. Compiling results into a single overall table automatically upon conclusion of all events
  9. Immediately emailing the results in the form of a spreadsheet to a mailing list
  10. Providing a simple interface to fetch the counts of the number of refreshments required
  11. Providing a simple interface for the Certification team to write all certificates with all the necessary data
  12. Minimizing the need for eventheads having to manually run around and manage the event by compiling all the necessary tools in one place
  13. Accounting for events with preliminaries before the complete festival, and updating the participant attendance from that interface, which doesn't have an effect on the refreshment orders as that is a separate table
